# 掲載追加・修正の手順

## 掲載候補の条件

次のいずれかを含む公開ページを対象とします。

- 個人タクシー開業の資格・手続
- 法令試験の勉強会・経営者学校
- 過去問・模擬試験・独自教材・学習アプリ
- 申請書類作成、面談、譲渡譲受の支援
- 開業者の体験記や合格実績

法人タクシー運転者の求人だけを目的とするページは原則として対象外です。

## 修正時の確認事項

1. URLを実際に開けるか
2. ページの運営団体名
3. 現在も開業希望者向け情報があるか
4. 地理試験など旧制度表記が残っていないか
5. `verified` を確認日に更新したか

## GitHub Issueから知らせる

リポジトリの **Issues** → **New issue** → **リンクの追加・修正** を利用してください。
