# 全国 個人タクシー開業・受験支援リンク集

全国の個人タクシー組合・支部・協会が公開している、開業希望者・受験者向けページを検索できる静的サイトです。
GitHub Pagesへそのまま公開できます。

## 主な機能

- キーワード検索
- 地域・団体系統・掲載内容で絞り込み
- 勉強会、過去問、アプリ、教材、申請支援、譲渡譲受などの特徴表示
- 旧制度表記やHTTPサイトへの注意表示
- URLコピー
- スマートフォン対応
- 週1回のリンク確認用GitHub Actions

## GitHub Pagesで公開する方法

1. このフォルダの中身を、GitHubの新しいリポジトリへアップロードします。
2. リポジトリの **Settings** → **Pages** を開きます。
3. **Build and deployment** の Source を **Deploy from a branch** にします。
4. Branchを **main**、フォルダを **/(root)** にして **Save** を押します。
5. 数分後に表示されるURLを開きます。

## 掲載情報を更新する方法

編集するファイルは基本的に `data/sites.json` だけです。

既存データを1件コピーして、以下を書き換えます。

```json
{
  "id": "重複しない英数字",
  "name": "ページ名",
  "organization": "組合・支部・協会名",
  "region": "関東",
  "prefecture": "千葉県",
  "system": "地域組合・独立系",
  "category": "勉強会・学校",
  "url": "https://example.jp/",
  "summary": "ページの特徴を短く記載します。",
  "features": ["勉強会・学校", "過去問"],
  "caution": "必要な場合だけ注意書きを記載します。",
  "verified": "2026-07-15",
  "priority": 80
}
```

- `caution` が不要な場合は、その行自体を削除できます。
- `priority` の数字が小さいほど「おすすめ順」で上に表示されます。
- `verified` は実際にページを開いて確認した日です。
- JSONの各項目の末尾にあるカンマの付け忘れ・付け過ぎに注意してください。

## パソコンで動作確認する方法

`index.html` を直接開くと、ブラウザの制限でJSONを読み込めない場合があります。
VS Codeの Live Server、または次のコマンドを利用してください。

```bash
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開きます。

## 自動リンク確認

`.github/workflows/link-check.yml` が毎週月曜日に各URLを確認します。
結果はGitHubの **Actions** から確認できます。古いHTTPサイトやアクセス制限のあるサイトは、実際には閲覧可能でも警告になる場合があります。

## 掲載方針

- 組合・支部・協会等が公開する開業希望者向けページを優先
- ブランドや系列を限定しない
- 専用ページが見つからない地域は公式団体一覧で補完
- 地理試験時代などの旧制度情報は、削除せず注意表示を付けて資料価値を残す
- 資格・試験・申請の最終確認は、管轄運輸局の最新公示を優先
