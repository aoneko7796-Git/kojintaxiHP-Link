# GitHub Pages root直置き版

このフォルダ内のファイルを、GitHubリポジトリの一番上（root）へすべてアップロードしてください。
フォルダを作る必要はありません。

必要なファイル:
- index.html
- styles.css
- app.js
- sites.json
- 404.html
- .nojekyll

公開設定:
Settings → Pages → Deploy from a branch → main → /(root) → Save

掲載情報は `sites.json` に保存されています。
`system` には、全国共通の分類として「でんでん系」「ちょうちん系」などを指定します。

## 分類方法

- 団体系統（大分類）：でんでん系／ちょうちん系／独立系／協会・広域団体／企業系・提携型
- 地域・支部（中分類）：西東京、千葉、練馬、札幌、大阪など
- 全国ブロック：関東、近畿、九州・沖縄など

地域・支部の候補は、選択した全国ブロックと団体系統に合わせて自動的に絞り込まれます。
